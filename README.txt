Sprockets v.0.1.0

Sprockets.js is created as a completely unobtrusive extension to the native types in javascript; it will not conflict with any other libraries that extend the native types.
This library is still in ALPHA.  It is not feature complete.

Features: 
	- Safe, unobtrusive native object extension.
	- String formatting.
	- Array extensions.
	- Guid objects.
	- Disposable objects and DOM objects.
	- Observable decorators.
	- Proxy classes
	- Memento pattern
	- Object pools.
	- disposable async Function pattern
